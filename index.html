Quiz Quartet

This is an fork of Vox's quiz generator tool. This version is designed to be used in conjunction with a server which serves individual, isolated pages and generates iframe code for them to be embedded in sites which cannot support JQuery or the additional imports required to run the quiz system. For more information, contact Nathan@NathanLawrence.org.

Instructions

Instructions
  1. Follow the steps below.
  2. Paste the embed code into a blank plain text document and save it with a unique, descriptive name, avoiding spaces or capital letters (eg. "quiz-mayday-trivia.html"). Please note that while Notepad on Windows will save documents as plain text, you may need to use a third party text editor, like the free TextWrangler.
  3. Upload it to the server using the included embed code generator package.
  4. Paste the generated embed code into your CMS and enjoy the quiz!
